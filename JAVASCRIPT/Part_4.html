Welcome to Part 9: Your JavaScript Level One Project!

For this project you will be building website that will seem to ask much questions to users,but you are secretly looking to spof anyone visiting your website will be asked a series of questions,only the spy will be able to give specific answers you expect. if all questions are answered you will post a secret message in console for the spy to read

Here are the four conditions you will check for the spy

  • The Spy has the same first letter of her Name and Last Name
  • The Spy is between the Age of 20 and 30 [inclusive of 20 and 30]
  • The Spy is at least 170 centimeters tall
  • The Spy has a pet name that ends with the letter "Y".

We are going to leave this project a little more open-ended,but if you feel like you need more guidance,check out the solution walkthrough videos. But believe it or not. we've actually covered everything you need to know to complete this task, i would suggest you use JavaScript to prompt the user for the information and set these to variables. Then use those variables to check for the four conditions to all be true. it's up to you what the secret message is!

A Spy with the following information shoud see your secret message:

  • Jose Johson
  • 26 years old
  • 175 cm tall
  • Pet name is "Sammy"

Keep in mind you can ask the question prompts in whatever order you want, you can even separate asking for the first name and then another prompt for last name

Also remember to switch out the script src to your own .js file! Right now it's connected to part4.js which contains the solutions
